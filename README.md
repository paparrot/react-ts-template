- [x] redux-toolkit
- [x] jest
- [x] postcss
- [x] css-modules

## Installation

```bash

# clone the repo
git clone https://github.com/paparrot/react-ts-bundle.git
cd react-ts-bundle

# install dependencies
yarn install

# run development server
yarn dev

```
